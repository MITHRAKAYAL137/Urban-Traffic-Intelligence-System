# Urban Traffic Intelligence System

An AI-powered traffic monitoring and prediction system built using Streamlit and Machine Learning.

## Features

- Traffic analytics dashboard
- Traffic density heatmap
- Road network simulation
- Real OpenStreetMap traffic map
- AI traffic prediction
- 24-hour traffic forecast

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- NetworkX
- Scikit-learn
- Folium

## Installation

git clone https://github.com/MITHRAKAYAL137/Urban-Traffic-Intelligence-System

cd smart-traffic-ai

pip install -r requirements.txt

streamlit run app.py
